About this project

Project Background

I am intern at Bhutan Data Scientist Pvt.Ltd. (BDS),Working on financial resilience at the household level in urban Bhutan.

This project analyzes real household data to understand income, expenses, savings behaviur, and financial stresspoints among Bhutanese Families.

Objectives

  • Identify key financial challenges in urban Bhutanese households
  • Visualize income, expenses, and saving patterns
  • Provide practical, data-informed saving and budgeting recommendations
  • Promote long-term financial stability and resilienece

Tools and Technology

Data analysis -> Tableau for visualization

Websites -> HTML, CSS, JavaScript (single-page application style)

Goal -> Make complex financial insights simple and actionable for everyone.
